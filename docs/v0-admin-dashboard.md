# V0 프롬프트: 관리자 대시보드

## 프롬프트

여행사 운영진용 관리자 대시보드를 만들어줘. Supabase 백엔드와 Clerk 인증을 활용해야 해.

### 1. 대시보드 메인 통계
- 실시간 예약 현황:
  - 전체 인원 / 최대 인원 (약 150명)
  - 남은 좌석 수
  - 예약률 (%)
  - 예상 수익 (인원 × 1인 요금)
- 싱글룸 예약 현황
- 최근 예약 내역 (테이블: 이름, 예약일, 인원, 상태)

### 2. 예약 관리 섹션
- 전체 예약 목록 (테이블)
  - 예약 번호
  - 예약자명
  - 연락처
  - 인원수
  - 객실 유형
  - 예약 상태 (예약완료, 결제대기, 취소 등)
  - 예약 날짜
  - 액션 버튼 (수정, 취소, 세부사항)
- 필터링: 예약 상태, 날짜 범위, 검색
- 엑셀 다운로드 기능

### 3. 여행 일정 관리
- 각 Day별 현황
- 숙박 호텔 정보 및 체크인/아웃 일정
- 식사 정보 (각 끼니별 인원, 식이제한 정보)
- 관광지 방문 일정 (시간, 입장료 총액 등)
- 가이드 및 차량 배정 정보
- 수정/추가 버튼

### 4. 재무 관리
- 매출 현황
  - 총 매출
  - 예수금 / 미수금
  - 환불액
- 비용 관리
  - 항공료
  - 호텔료
  - 가이드 비용
  - 차량 비용
  - 입장료 등
- 수익성 분석 차트

### 5. 고객 커뮤니케이션
- 예약자 메시지 수신 내역
- 일괄 메일/SMS 발송 기능
  - 예약확인 안내
  - 출발 전 안내사항
  - 특별 공지사항
- 통지 템플릿 관리

### 6. 데이터 분석
- 월별/주별 예약 추이 그래프
- 객실 유형별 예약 비율 파이차트
- 고객 만족도 데이터 (리뷰 기반)
- 리피터율 분석

### 7. 시스템 설정
- 여행 상품 정보 수정
  - 출발일, 귀국일
  - 가격 설정
  - 최대 인원
  - 포함/불포함 사항
- 사용자 계정 관리 (Clerk 연동)
  - 관리자 추가/제거
  - 권한 설정
- 공지사항 및 배너 관리

### 8. 사용자 접근 제어
- 로그인: Clerk 인증
- 역할 기반 접근 (예: 운영진만 가능)
- 로그아웃 / 세션 관리

## 디자인 요구사항

- 통계 대시보드형 차트 (Pie, Line, Bar 등)
- 예약/인원/객실 테이블 정렬/필터
- 하이라이트 컬러: Mint, Navy, Gold
- Sidebar + Main Panel 레이아웃
- 반응형 디자인 / 모바일 최적화

## 기술/구현 스택
- React/Next.js
- TypeScript
- Tailwind CSS
- Clerk (인증)
- Supabase (DB, 실시간 데이터)
- Chart.js 또는 Recharts
